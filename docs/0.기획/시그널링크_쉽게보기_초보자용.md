## 이 앱, 한 줄로

**주식이 왜 오르고 내렸는지를, 어려운 경제 용어 대신 초보자도 아는 말로 설명해 주는 앱**이에요.

| 누구를 위해 | 언제 | 비용 | 만드는 사람 |
|------------|------|------|------------|
| 2030 직장인·주린이 | 출근길 3분 시황 파악 | 월 5만 원 이내 | 1인 개발 · 13~15주 |

## 무슨 문제를 푸나 — "번역"이 핵심

같은 상황인데, 숫자만 나열된 화면을 **이야기**로 바꿔줍니다.

**❌ 보통 증권사 화면**
```
삼성전자 −3.2% · 외국인 순매도 1,240억 · 프로그램 매물 출회 · KOSPI −1.4% …
```
→ 초보자: *"그래서 왜 떨어졌다는 거지?"*

**✅ 시그널링크**
> "오늘 하락은 **대부분 시장 전체가 빠진 영향**이에요. 외국인이 많이 팔았고, 거래는 평소의 2배였어요. 회사에 특별한 악재 공시는 없었어요."

→ 초보자: *"아, 이 종목만의 문제는 아니구나."*

## 만드는 기능 4가지

가장 중요한 건 **① 특징주 Why 보드**예요. 나머지 셋은 그걸 더 잘 이해하게 돕는 도구입니다.

| 기능 | 한 줄 설명 | |
|------|-----------|---|
| **F5. 특징주 Why 3줄 보드** | 오늘 많이 움직인 종목을 골라, "왜 그랬는지"를 딱 3줄로. "이 악재 오래 갈까?" 판단 재료(체력 진단)도 함께. | ⭐ **핵심** |
| **F1. 매크로 도미노 시뮬레이터** | 금리·물가 같은 큰 사건이 어디로 번지는지 도미노처럼. "이론 vs 실제 과거"도 비교. | |
| **F3. 경제 신호등 + 시장 날씨** | 경제 지표 발표를 🟢좋음 🟡애매 🔴나쁨 신호로. 시장 분위기는 '화창~폭풍' 날씨로. | |
| **F2. 금융 번역기** | 어려운 용어에 점선이 그어지고, 터치하면 쉬운 뜻풀이가 뜸. | |

## 핵심 기능은 어떻게 동작하나 — "왜 올랐지?"를 알아내는 4단계

먼저 매일 데이터를 모으고, 그걸 4단계로 걸러 카드 3줄을 만듭니다.
AI는 **이미 검증된 데이터를 통역만** 해요 (없는 얘기를 지어내지 않게).

```
① 매일 데이터 모으기
   [주가·거래량] [외국인·기관 수급] [DART 공시] [네이버 뉴스]
                        │
                        ▼
┌─────────────────────────────────────────────────────────┐
│  Why 엔진 4단계                                            │
│                                                           │
│  1. 시장 탓? 이 종목만의 일?                               │
│     → 등락에서 '시장이 움직인 몫'(베타 × 지수)을 빼본다     │
│                                                           │
│  2. 같은 업종이 다 같이 움직였나?                          │
│     → 업종 전체 이슈인지 확인                               │
│                                                           │
│  3. 무슨 일이 있었나 (근거)                                │
│     → 공시·뉴스에서 이유 후보 최대 3건 (공시를 먼저!)       │
│                                                           │
│  4. 누가 사고 팔았나 (수급)                                │
│     → 외국인·기관 순매수 방향 (장중엔 '잠정' 표시)          │
└─────────────────────────────────────────────────────────┘
                        │
                        ▼
   → Why 카드 (뉴스 있으면 AI 통역 3줄, 없으면 템플릿 문장)
```

> 🚫 **절대 규칙**: "사세요 / 파세요 / 목표가 얼마" 같은 조언은 **절대 안 함**.
> 무슨 일이 있었는지 **'사실'만** 설명하고, 판단은 사용자가 합니다. (프롬프트 + 출력 필터로 이중 차단)

## Why 카드, 실제로 이렇게 보여요

기획서에 나온 예시 카드예요. **①무슨 일 ②수급 ③맥락**, 딱 3줄.

```
📉 한화오션  −8.2%   [잠정]
────────────────────────────────────────
① 무슨 일 : 오후 2시 잠수함 수주 불발 공시 → 원문 링크
② 수급    : 기관 −30만 주 매도 주도 · 거래량 평소 4배 · 개인 +25만 주
③ 맥락    : 이 중 −2%는 시장 전체가 빠진 몫
────────────────────────────────────────
🔍 장기 체력 진단 펼쳐보기 ▾     본 정보는 투자 조언이 아닙니다
```

## 속은 어떻게 생겼나 — "창고 5개 + 일꾼 2명"

데이터를 종류별로 **5개 창고(도메인)**에 나눠 담고, **일꾼 2명**이 각각 모으고(collector) 내어줍니다(api).
창고끼리는 서로 함부로 못 건드리게 벽이 쳐져 있어요(경계를 코드로 강제).

```
      [시세 창고]        [이슈 창고]        [거시 창고]
     domain-market      domain-issue      domain-macro
     주가·수급·베타      공시·뉴스         지표·신호등·날씨
          │                 │                 │
          └────────┬────────┴─────────────────┘
                   ▼  (읽기만, 고치진 않음)
            🫀 [Why 엔진 — 심장]
              domain-insight
              4단계 조립 → Why 카드 자동 생성

      [콘텐츠 창고]                  [일꾼 2명]
      domain-content                app-collector : 30분마다 데이터 수집
      용어사전·도미노 룰(독립)        app-api       : 화면에 데이터 제공
```

**지금 어디까지 됐나** (개발 현황)

| 창고/일꾼 | 상태 | 설명 |
|-----------|------|------|
| 시세 창고 `market` | 🟢 완료 | 모든 계산의 재료 |
| 이슈 창고 `issue` | 🟢 완료 | 급등락의 이유 후보 |
| **Why 엔진 `insight`** | 🟡 진행중(~70%) | 심장. AI 통역 붙이는 단계만 남음 |
| 거시 창고 `macro` | ⚪ 예정 | 신호등·날씨 |
| 콘텐츠 창고 `content` | ⚪ 예정 | 용어사전·도미노 |
| 일꾼 `collector`/`api` | 🟢/⚪ | 수집은 대부분 완료, 조회 API는 예정 |

## 믿을 수 있으려면 지키는 원칙 3가지

1. 🙅 **조언은 안 해요** — 사실만. "외국인이 순매수했다"(O) / "외국인이 사니 오를 것"(X)
2. 🕒 **실시간이 아니에요** — 30분마다 갱신. 초 단위 트레이딩용이 아니라 '상황 이해용'
3. 🔁 **잠정 → 확정** — 장중 숫자는 '잠정' 표시, 장 마감 뒤 확정치로 덮어씀

**빠른 사실**: 기간 13~15주 · 월 5만 원 이내 · AI는 Gemini(일 20~30건만) · 서버 1대(컨테이너 2개) · 모바일 우선(390px)

## 자주 나오는 용어 미리 풀어보기

앱의 '금융 번역기'가 할 일을 여기서 몇 개만 미리.

| 용어 | 쉬운 뜻 |
|------|---------|
| **특징주** | 오늘 유독 많이 움직인(±5%↑) 또는 거래가 폭증한(평소 3배) 종목 |
| **수급** | 누가(개인·외국인·기관) 얼마나 사고팔았는지. 방향으로 분위기를 봄 |
| **공시** | 회사가 법적으로 알리는 공식 발표. 뉴스와 달리 '사실'이라 근거로 먼저 씀 |
| **베타(β)** | 시장이 1% 움직일 때 이 종목이 보통 몇 % 움직이는지. 시장 영향 몫 계산에 사용 |
| **거래대금** | 사고판 '금액'의 크기. 클수록 사람들이 주목한 종목이라 위로 올려 보여줌 |
| **도미노 룰** | "금리 오르면 → 채권은…"처럼 사건이 번지는 경로를 정리한 규칙(예외 사례 포함) |
