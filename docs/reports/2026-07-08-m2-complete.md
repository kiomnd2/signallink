## 3. 아키텍처·검증
- **헥사고날 Level 1** 일관 적용: 5개 도메인 모듈 내부 `domain/application(port·service)/adapter.out`. 의존 방향 빌드 강제.
- **KIS 정책 준수**: ① `RateLimiter(1.0)`(실측 초당 1건) · ② `KisTokenProvider` DB 캐시 재사용 · ③ 100영업일 페이지네이션(백필).
- **테스트**: 도메인 규칙·파서(실응답 픽스처)·수집 서비스(포트 스텁) 단위 + Testcontainers Postgres 영속(로컬 skip·CI 실행). PR #1~#16 전부 CI green.
- **KIS 실검증**: 토큰·순위·수급 2종·일별시세·지수·종목마스터 실호출 확인. (사고: `.env` 따옴표 → 제거, PR#7 누락 커밋 → cherry-pick 복구.)

## 5. 다음 마일스톤 — M3 Why 엔진
원천 데이터가 준비되어 M3 착수 가능:
특징주 선정(거래대금 상위 ±5%/거래량 3배) → 시장분해(등락률 − β×지수) → 수급 진단 → 이슈 매칭(공시 우선·뉴스 시간창) → LLM 요약(금지어 필터·폴백). **최다 리스크 구간**이므로 백필 데이터로 과거 재현 검증하며 진행.
