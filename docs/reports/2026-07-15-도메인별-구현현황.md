## 1. `common` — ✅ 완료 (인프라 기반)

**구현됨**
- `ApiResponse`(면책 문구 고정), `TimeUtils`(KST / nowKst / todayKst)

- `DiscordNotifier`(잡 실패 알림), 테스트 2건(`ExternalApiClientTest`·`RateLimiterTest`)

**미구현/잔여**: 없음. 전 모듈이 이 위에서 동작 중.

## 2. `domain-market` — ✅ 완료 (M2)

**구현됨** — 6개 테이블 적재 경로 전부
- **종목 마스터**(J1): `StockMasterSyncService` + `StockMasterParser`(`.mst`, ST·6자리 필터)
- **일봉/지수**: `PriceCollectService` + `PriceBackfillService`(2년 백필, 100영업일 페이지네이션), `DailyMarketCollectService`
- **투자자 수급**: `InvestorFlowCollectService` — 장중 가집계(FHPTJ04400000) + 마감 확정(FHKST01010900) 이원화, 잠정→확정 upsert
- **파생 지표**: `BetaCalculator`(60일 Cov/Var → `StockBetaService`), `VolumeRatioCalculator`(vol_ratio_20d)
- KIS 클라이언트 4종(`KisMarketClient`·`KisInvestorClient`·`StockMasterFileClient` 등) + `KisTokenProvider`(DB 캐시), `RateLimiter(1.0)` 실측 반영
- 테스트: 도메인 규칙·파서(픽스처)·서비스(포트 스텁)·영속(Testcontainers) 풀세트

**미구현/잔여**
- `stock.sector_code`(업종)가 아직 nullable — `.mst` tail 지수업종 오프셋 미확정. **M3 업종판별에 필요** (BACKLOG 등재)
- `KisClient` 공통 베이스 추출(중복 제거, BACKLOG)
- 실호출 dry-run(`BACKFILL_ENABLED=true`로 로컬 DB 소수 종목 실적재 확인)

## 4. `domain-insight` — 🔄 진행중 (~70%, 서비스의 심장)

**구현됨** — 3-A · 3-B · 3-C · 3-D · 3-F
- **3-A 특징주 선정**: `FeatureStockSelector` + `FeatureStockSelectService` (±5%/거래량 3배 → 거래대금 상위)
- **3-B 시장분해**: `MarketDecomposer`(등락률 − β×지수), `SectorSyncEvaluator`(업종 동조), `WhatHappenedComposer`(템플릿 문장)
- **3-C 수급 진단**: `FlowDiagnoser` + `FlowDiagnoseService`
- **3-D 이슈 매칭**: `IssueMatcher` + `IssueMatchService` (공시 우선·뉴스 시간창·후보 3건)
- **3-F J2 파이프라인**: `FeatureCardPipelineService` — 선정→분해→진단→매칭→`feature_card` upsert, **종목 단위 예외 격리**. `source_refs`는 어댑터에서 jsonb 수동 직렬화(`SourceRefsJson`). `FeatureCardJob`(장중 30분) 배선 + Discord 실패 알림

**미구현 (핵심 잔여)**
- ⬜ **3-E LLM 클라이언트** — Gemini 호출·JSON 파싱·**금지어(조언성) 필터**·템플릿 폴백·일 상한 가드. 현재 `what_happened`는 **템플릿 폴백**이며 `llm_used=false` 고정. 배선점은 파이프라인의 what_happened 생성부. (Gemini 키 대기 — BACKLOG #4)
- ⬜ **3-G health_check** — `insight.health_check` 테이블은 있으나 엔티티·서비스 전무. 이슈 성격 태그·DART 재무 추세·flow_after 추적(J6)
- ⬜ **3-H macro 연계** — `macro_event` 기반 context_note 보강 (domain-macro 미착수에 종속)
