## 1. `common` — 공통 인프라 (전 모듈 의존)

### 역할·경계
프레임워크·외부 연동의 공통 토대. 외부 API 호출 규율(유량 제한·재시도), KST 시간, 표준 API 응답(면책 문구 고정), 운영 알림(Discord)을 제공한다. 도메인 지식은 없다.

### 주요 기능
| 컴포넌트 | 역할 |
|----------|------|
| `RateLimiter` | 초당 N건 블로킹 유량 제한 (KIS 실측 1건/초 → `RateLimiter(1.0)`) |
| `ExternalApiClient` | 외부 API 베이스 — rate limit + 지수 백오프 재시도(429/5xx/IO만) |
| `TimeUtils` | KST 고정 시간 (`KST`·`nowKst`·`todayKst`) |
| `ApiResponse` | 표준 응답 래퍼 + 투자 면책 문구 고정 |
| `DiscordNotifier` | 잡 실패 알림 (webhook 없으면 no-op) |

### 핵심 코드

**`RateLimiter` — 스레드 안전 최소 간격 보장**
```java
public synchronized void acquire() {
    long now = System.nanoTime();
    long waitNanos = nextFreeAt - now;
    nextFreeAt = Math.max(now, nextFreeAt) + minIntervalNanos;
    if (waitNanos > 0) {
        Thread.sleep(waitNanos / 1_000_000, (int) (waitNanos % 1_000_000));
    }
}
```

**`ExternalApiClient` — 재시도 판정과 지수 백오프(지터 포함)**
```java
protected <T> T execute(String opName, Supplier<T> call) {
    for (int attempt = 1; attempt <= maxAttempts; attempt++) {
        rateLimiter.acquire();
        try {
            return call.get();
        } catch (RuntimeException e) {
            if (!isRetryable(e) || attempt == maxAttempts) throw e;
            sleep(backoffMillis(attempt)); // base * 2^(n-1) + jitter
        }
    }
}
private boolean isRetryable(RuntimeException e) {
    if (e instanceof ResourceAccessException) return true;      // 타임아웃·커넥션
    if (e instanceof RestClientResponseException re) {
        int s = re.getStatusCode().value();
        return s == 429 || s >= 500;                            // 나머지 4xx는 재시도 안 함
    }
    return false;
}
```

**`BetaCalculator` — β = Cov(종목수익률, 지수수익률) / Var(지수수익률)**
```java
public static BigDecimal beta60d(List<DatedClose> stockCloses, List<DatedClose> indexCloses) {
    // 공통 거래일로 정렬 → 일수익률 배열 산출
    List<DatedClose> aligned = stockCloses.stream()
        .filter(s -> indexByDate.containsKey(s.date()))
        .sorted(Comparator.comparing(DatedClose::date)).toList();
    if (aligned.size() < MIN_RETURNS + 1) return null;      // 수익률 20개 미만이면 계산 불가

    double varIndex = variance(indexReturns);
    if (varIndex == 0.0) return null;                       // 지수 분산 0이면 계산 불가
    double beta = covariance(stockReturns, indexReturns) / varIndex;
    return BigDecimal.valueOf(beta).setScale(3, RoundingMode.HALF_UP);
}
```

## 4. `domain-insight` — 특징주·Why 엔진 (서비스의 심장) 🔄 진행중 (~70%)

### 핵심 코드

**3-A `FeatureStockSelector` — 트리거(±5% or 거래량 3배) → 거래대금 상위 40**
```java
public static final BigDecimal SURGE_PCT = new BigDecimal("5.0");
public static final BigDecimal VOLUME_MULT = new BigDecimal("3.0");
public static final int DEFAULT_TOP_N = 40;

public static List<FeatureCandidate> select(List<StockDaySnapshot> snapshots, LocalDate d, int topN) {
    return snapshots.stream().map(s -> classify(s, d)).filter(Objects::nonNull)
        .sorted(Comparator.comparingLong(FeatureCandidate::tradingValue).reversed()) // 거래대금 상위
        .limit(topN).toList();
}
private static FeatureCandidate classify(StockDaySnapshot s, LocalDate d) {
    boolean surge  = s.changeRate() != null && s.changeRate().abs().compareTo(SURGE_PCT) >= 0;
    boolean volume = s.volRatio20d() != null && s.volRatio20d().compareTo(VOLUME_MULT) >= 0;
    if (!surge && !volume) return null;                    // 트리거 미해당 → 제외
    return new FeatureCandidate(s.stockCode(), d, s.changeRate(),
                                surge ? TriggerType.SURGE : TriggerType.VOLUME, s.tradingValue());
}
```

**3-B `MarketDecomposer` — 등락을 "시장 몫"과 "개별 몫"으로 분해**
```java
// marketContrib = β × 지수등락률(시장으로 설명되는 몫), excess = 등락률 − marketContrib(개별 몫)
public static MarketDecomposition decompose(BigDecimal changeRate, BigDecimal beta, BigDecimal indexChangeRate) {
    if (beta == null || indexChangeRate == null)           // β·지수 없으면 전부 개별로
        return new MarketDecomposition(null, changeRate, Driver.STOCK_SPECIFIC);
    BigDecimal contrib = beta.multiply(indexChangeRate).setScale(3, HALF_UP);
    BigDecimal excess  = changeRate.subtract(contrib).setScale(3, HALF_UP);
    Driver driver = contrib.abs().compareTo(excess.abs()) >= 0  // 더 큰 쪽이 주된 동인
        ? Driver.MARKET_DRIVEN : Driver.STOCK_SPECIFIC;
    return new MarketDecomposition(contrib, excess, driver);
}
```

**3-C `FlowDiagnoser` — 외국인·기관 순매수만, 1억 미만은 중립, 조언 금지**
```java
public static final long MIN_MEANINGFUL_MILLION = 100; // 1억 원

public static FlowDiagnosis diagnose(Long foreignMillion, Long orgMillion, boolean provisional) {
    if (foreignMillion == null && orgMillion == null)
        return new FlowDiagnosis("외국인·기관 수급 데이터가 아직 없어요.", provisional);
    int foreign = direction(foreignMillion), org = direction(orgMillion); // +1/-1/0
    String prefix = provisional ? "장중 잠정 집계로는, " : "";
    String body = (foreign == 0 && org == 0)
        ? "외국인·기관 모두 뚜렷한 매매는 없었어요."
        : actor("외국인", foreign, foreignMillion) + ", " + actor("기관", org, orgMillion) + "."
          + nuance(foreign, org); // 동반 매수/매도 or 엇갈림
    return new FlowDiagnosis(prefix + body, provisional);
}
private static int direction(Long m) {          // 소액(<1억)·null은 중립
    return (m == null || Math.abs(m) < MIN_MEANINGFUL_MILLION) ? 0 : (m > 0 ? 1 : -1);
}
```
