## M0 실측 후속 이슈 (2026-07-08 검증에서 발견)

| 항목 | 내용 | 후속 |
|------|------|------|
| KIS 일별 총량 제한 | probe는 초당 한도(1건)만 확인, **일별 총 호출 제한 여부 미확인** | 운영 중 모니터. J2 일 호출 ~1,000여 콜 추정 → 대부분 무난할 것 |
| 외인·기관 가집계 장중 확인 | `FHPTJ04400000` 09:22 실행 시 0행(첫 갱신 전) | 장중 09:30~14:30에 재실행해 실데이터·갱신 주기 확인 |
| RateLimiter 설정값 | 실측 초당 1건 → M2 KIS 클라이언트에서 `RateLimiter(1.0)`(안전 0.8) | M2 market 구현 시 적용 |
| Python CA 번들 미보유 | 이 머신 Python 3.14가 CA 미보유 → `SSL_CERT_FILE=/etc/ssl/cert.pem` 필요 | M0 문서 §4에 기록. 필요 시 `certifi` 설치로 상시화 |
| `.env.example` 변수명 | `KIS_APP_KEY`→`KIS_APPKEY`로 스크립트/문서와 정렬함 | 커밋 반영 필요 |
| ⚠️ KIS 토큰 재발급 제한 발동 (2026-07-08) | 검증 중 파일 캐시(`~/.kis_token.json`) 삭제 + 반복 발급 시도 → tokenP HTTP 403(제한/IP 차단 가능) | ~1시간 대기 후 **1회만** 발급. **토큰 캐시 절대 삭제 금지** — 앱 `KisTokenProvider`는 DB 캐시라 재발급 안 함(무관). 슬라이스 2 TR 검증은 회복 후 |

## 아이디어 (1차 범위 밖)

- **미국증시(해외주식) 확장** — 2차 검토. **제약**: (1) 미국은 외국인/기관 **일별 순매수(수급) 데이터가 없음** → 제품 핵심 기둥이 미적용, (2) 공시=SEC EDGAR·뉴스=영문 → **이슈 파이프라인 완전 별개**, (3) M2·M3 사실상 2배 + 실시간 시세 유료·장시간(23:30~06:00 KST)·환율(USD). **열림**: KIS 해외주식 API 존재 + 헥사고날 구조·`stock.market_type`으로 어댑터 추가 확장 가능(단 `investor_flow`는 국내 전용으로 분기 필요). → **국내 런칭 후 착수.**
